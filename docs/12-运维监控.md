## 2. 监控指标体系

### 2.1 核心指标定义

> **⚠️ 说明**：以下 `MetricsCollector` 是框架提供的**通用指标注册容器**。
> 框架本身不预设业务指标（如连接数、帧率），而是由业务代码在运行时注册所需的 Counter/Gauge/Histogram。
> `MemoryMonitor` 会在启动后自动注册 GC 相关指标。

```csharp
/// <summary>
/// 指标收集器（单例）
///
/// 管理所有指标的注册、查询、快照与重置。
///
/// 使用方式：
/// 1. 业务代码调用 RegisterCounter/RegisterGauge/RegisterHistogram 注册指标
/// 2. 获取指标实例后调用 Increment/Set/Record 更新值
/// 3. PrometheusExporter 定时调用 GetDetailedSnapshot 导出
/// </summary>
public class MetricsCollector
{
    // ============================================================
    // 单例
    // ============================================================

    public static MetricsCollector Instance { get; } = new();

    private volatile bool _enabled;

    // ============================================================
    // 指标注册
    // ============================================================

    /// <summary>
    /// 注册或获取计数器（只增不减）
    /// </summary>
    /// <param name="name">指标名称</param>
    /// <param name="description">描述</param>
    /// <returns>计数器实例；异常时返回 null</returns>
    public ICounter? RegisterCounter(string name, string description);

    /// <summary>
    /// 注册或获取量规（可增可减）
    /// </summary>
    /// <param name="name">指标名称</param>
    /// <param name="description">描述</param>
    /// <param name="valueProvider">动态取值委托；若为 null 则使用 Set 模式</param>
    public IGauge? RegisterGauge(string name, string description, Func<double>? valueProvider = null);

    /// <summary>
    /// 注册或获取直方图（统计分布）
    /// </summary>
    /// <param name="name">指标名称</param>
    /// <param name="description">描述</param>
    /// <param name="capacity">环形缓冲容量（默认 1024）</param>
    public IHistogram? RegisterHistogram(string name, string description, int capacity = 1024);

    // ============================================================
    // 指标查询
    // ============================================================

    /// <summary>查询指定名称指标的当前值</summary>
    public double? GetValue(string name);

    /// <summary>获取所有指标的当前值快照（名称 → 值）</summary>
    public IDictionary<string, double> Snapshot();

    /// <summary>获取所有指标的详细快照（包含类型、描述、值）</summary>
    public IEnumerable<MetricSnapshotEntry> GetDetailedSnapshot();

    /// <summary>获取所有已注册的指标名称</summary>
    public IEnumerable<string> GetRegisteredMetrics();

    // ============================================================
    // 按类型获取
    // ============================================================

    public ICounter? GetCounter(string name);
    public IGauge? GetGauge(string name);
    public IHistogram? GetHistogram(string name);

    // ============================================================
    // 控制
    // ============================================================

    public bool IsEnabled { get; }
    public void Enable();
    public void Disable();

    /// <summary>重置所有指标值（不改变启用状态）</summary>
    public void Reset();

    /// <summary>完全重置：清空所有指标并禁用（主要用于测试）</summary>
    public void ResetAll();
}

/// <summary>
/// 计数器接口（只增不减）
/// </summary>
public interface ICounter
{
    long Count { get; }
    void Increment();
    void IncrementBy(long value);
}

/// <summary>
/// 量规接口（可增可减）
/// </summary>
public interface IGauge
{
    double Value { get; }
    void Set(double value);
}

/// <summary>
/// 直方图接口（统计分布）
/// </summary>
public interface IHistogram
{
    double Value { get; }
    void Record(double value);
    int SampleCount { get; }
    double P50 { get; }
    double P95 { get; }
    double P99 { get; }
}

/// <summary>
/// 指标快照条目
/// </summary>
public class MetricSnapshotEntry
{
    public string Name { get; set; }
    public double Value { get; set; }
    public string Description { get; set; }
    public string Type { get; set; } // "counter" / "gauge" / "histogram"
}
```

## 3. 健康检查系统

### 3.1 健康检查服务

> **⚠️ 说明**：框架使用 `HealthCheckService`（单例），默认**不注册任何检查项**。
> 业务层通过 `Register()` 显式注册检查项，或使用扩展方法 `AddMemoryCheck` / `AddSessionCountCheck`。

```csharp
/// <summary>
/// 健康检查服务（单例）
///
/// 使用 "聚合最坏状态" 策略：
/// OverallStatus = 所有条目中最大的枚举值（Healthy < Degraded < Unhealthy）
/// </summary>
public class HealthCheckService
{
    public static HealthCheckService Instance { get; }

    // ============================================================
    // 注册检查项
    // ============================================================

    /// <summary>
    /// 注册一个异步健康检查项
    /// 委托返回 (Result, Description)，
    /// 若委托内部抛异常，会被捕获并标记为 Unhealthy。
    /// </summary>
    public void Register(
        string name,
        Func<CancellationToken, Task<(HealthCheckResult Result, string Description)>> checker);

    /// <summary>
    /// 注册一个同步健康检查项（内部转换为异步）
    /// </summary>
    public void Register(
        string name,
        Func<HealthCheckResult> syncChecker,
        string? description = null);

    // ============================================================
    // 执行检查
    // ============================================================

    /// <summary>
    /// 并行执行所有已注册的检查项，返回聚合报告
    /// </summary>
    public async Task<HealthCheckStatus> CheckHealthAsync(CancellationToken cancellationToken = default);

    /// <summary>便捷方法：执行检查并渲染为文本</summary>
    public async Task<string> RenderTextAsync(CancellationToken ct = default);

    /// <summary>便捷方法：执行检查并渲染为 JSON</summary>
    public async Task<string> RenderJsonAsync(CancellationToken ct = default);

    /// <summary>清空所有已注册的检查项（主要用于测试）</summary>
    public void Clear();
}

/// <summary>
/// 健康检查结果枚举
/// </summary>
public enum HealthCheckResult
{
    Healthy = 0,   // 健康
    Degraded = 1,  // 降级（部分异常）
    Unhealthy = 2  // 不健康（严重问题）
}

/// <summary>
/// 健康检查条目
/// </summary>
public class HealthCheckEntry
{
    public string Name { get; }
    public HealthCheckResult Result { get; }
    public string Description { get; }
}

/// <summary>
/// 健康检查聚合状态
/// </summary>
public class HealthCheckStatus
{
    public HealthCheckResult OverallStatus { get; }
    public HealthCheckEntry[] Entries { get; }
    public string RenderText();
    public string RenderJson();
}

/// <summary>
/// 健康检查服务扩展方法
/// </summary>
public static class HealthCheckServiceExtensions
{
    /// <summary>
    /// 注册内存使用量检查
    /// 阈值：< 75% → Healthy，75%~100% → Degraded，>= 100% → Unhealthy
    /// </summary>
    /// <param name="memoryThresholdBytes">内存阈值（字节），默认 2GB</param>
    public static void AddMemoryCheck(
        this HealthCheckService service,
        long memoryThresholdBytes = 2L * 1024 * 1024 * 1024);

    /// <summary>
    /// 注册活动会话数检查
    /// </summary>
    /// <param name="server">TcpServer 实例</param>
    /// <param name="degradedThreshold">降级阈值（默认 500）</param>
    /// <param name="unhealthyThreshold">异常阈值（默认 900）</param>
    public static void AddSessionCountCheck(
        this HealthCheckService service,
        TcpServer server,
        int degradedThreshold = 500,
        int unhealthyThreshold = 900);
}
```

### 3.3 常用健康检查实现

```csharp
/// <summary>
/// 连接数健康检查
/// </summary>
public class ConnectionHealthCheck : IHealthCheck
{
    public string Name => "connection";
    
    private readonly TcpServer _server;
    private const int MaxConnectionsWarning = 8000;
    private const int MaxConnectionsCritical = 9500;
    
    public ConnectionHealthCheck(TcpServer server)
    {
        _server = server;
    }
    
    public Task<HealthCheckResult> CheckAsync()
    {
        var connectionCount = _server.GetConnectionCount();
        
        var status = connectionCount < MaxConnectionsWarning 
            ? HealthCheckStatus.Healthy
            : connectionCount < MaxConnectionsCritical
                ? HealthCheckStatus.Degraded
                : HealthCheckStatus.Unhealthy;
        
        return Task.FromResult(new HealthCheckResult
        {
            Name = Name,
            Status = status,
            Message = $"当前连接数: {connectionCount}",
            Data = new Dictionary<string, object>
            {
                ["connection_count"] = connectionCount,
                ["max_connections"] = MaxConnectionsCritical
            }
        });
    }
}

/// <summary>
/// 帧率健康检查
/// </summary>
public class FrameRateHealthCheck : IHealthCheck
{
    public string Name => "framerate";
    
    private readonly GameLoop _gameLoop;
    private const double MinHealthyFps = 30.0;
    private const double MinDegradedFps = 20.0;
    
    public FrameRateHealthCheck(GameLoop gameLoop)
    {
        _gameLoop = gameLoop;
    }
    
    public Task<HealthCheckResult> CheckAsync()
    {
        var fps = _gameLoop.GetAverageFps();
        
        var status = fps >= MinHealthyFps
            ? HealthCheckStatus.Healthy
            : fps >= MinDegradedFps
                ? HealthCheckStatus.Degraded
                : HealthCheckStatus.Unhealthy;
        
        return Task.FromResult(new HealthCheckResult
        {
            Name = Name,
            Status = status,
            Message = $"平均帧率: {fps:F1} Hz",
            Data = new Dictionary<string, object>
            {
                ["average_fps"] = fps,
                ["target_fps"] = _gameLoop.TargetFps
            }
        });
    }
}

/// <summary>
/// 队列积压健康检查
/// </summary>
public class QueueBacklogHealthCheck : IHealthCheck
{
    public string Name => "queue_backlog";
    
    private readonly MessageQueue _messageQueue;
    private const int WarningThreshold = 500;
    private const int CriticalThreshold = 2000;
    
    public QueueBacklogHealthCheck(MessageQueue messageQueue)
    {
        _messageQueue = messageQueue;
    }
    
    public Task<HealthCheckResult> CheckAsync()
    {
        var queueLength = _messageQueue.Count;
        
        var status = queueLength < WarningThreshold
            ? HealthCheckStatus.Healthy
            : queueLength < CriticalThreshold
                ? HealthCheckStatus.Degraded
                : HealthCheckStatus.Unhealthy;
        
        return Task.FromResult(new HealthCheckResult
        {
            Name = Name,
            Status = status,
            Message = $"队列长度: {queueLength}",
            Data = new Dictionary<string, object>
            {
                ["queue_length"] = queueLength,
                ["warning_threshold"] = WarningThreshold,
                ["critical_threshold"] = CriticalThreshold
            }
        });
    }
}

/// <summary>
/// 数据库连接健康检查
/// </summary>
public class DatabaseHealthCheck : IHealthCheck
{
    public string Name => "database";
    
    private readonly IDatabase _database;
    private const int TimeoutMs = 5000;
    
    public DatabaseHealthCheck(IDatabase database)
    {
        _database = database;
    }
    
    public async Task<HealthCheckResult> CheckAsync()
    {
        var sw = Stopwatch.StartNew();
        
        try
        {
            var canConnect = await _database.PingAsync(TimeoutMs);
            sw.Stop();
            
            var status = canConnect 
                ? HealthCheckStatus.Healthy 
                : HealthCheckStatus.Unhealthy;
            
            return new HealthCheckResult
            {
                Name = Name,
                Status = status,
                Message = canConnect ? "数据库连接正常" : "数据库连接失败",
                Duration = sw.Elapsed,
                Data = new Dictionary<string, object>
                {
                    ["latency_ms"] = sw.ElapsedMilliseconds
                }
            };
        }
        catch (Exception ex)
        {
            sw.Stop();
            
            return new HealthCheckResult
            {
                Name = Name,
                Status = HealthCheckStatus.Unhealthy,
                Message = $"数据库异常: {ex.Message}",
                Duration = sw.Elapsed
            };
        }
    }
}
```

### 3.3 健康检查接口使用示例

```csharp
// 启动健康检查端点（零依赖，基于 HttpListener）
var healthEndpoint = new HealthEndpoint(port: 8080);
healthEndpoint.Start();

// 注册内存检查（使用扩展方法）
HealthCheckService.Instance.AddMemoryCheck(memoryThresholdBytes: 1024L * 1024 * 1024); // 1GB

// 注册会话数检查
HealthCheckService.Instance.AddSessionCountCheck(
    server: tcpServer,
    degradedThreshold: 500,
    unhealthyThreshold: 900);

// 自定义检查项
```

### 5.2 优雅关闭实现

```csharp
/// <summary>
/// 优雅关闭管理器
/// 
/// 职责：
/// 1. 处理关闭信号
/// 2. 分阶段执行关闭
/// 3. 超时控制
/// 4. 状态报告
/// </summary>
public class GracefulShutdownManager
{
    // ============================================================
    // 配置
    // ============================================================
    
    /// <summary>
    /// 等待消息队列清空的超时时间
    /// </summary>
    private static readonly TimeSpan QueueDrainTimeout = TimeSpan.FromSeconds(30);
    
    /// <summary>
    /// 等待请求处理完成的超时时间
    /// </summary>
    private static readonly TimeSpan RequestCompletionTimeout = TimeSpan.FromSeconds(60);
    
    /// <summary>
    /// 总关闭超时时间
    /// </summary>
    private static readonly TimeSpan TotalShutdownTimeout = TimeSpan.FromMinutes(2);
    
    // ============================================================
    // 状态
    // ============================================================
    
    /// <summary>
    /// 是否正在关闭
    /// </summary>
    private volatile bool _isShuttingDown;
    
    /// <summary>
    /// 关闭开始时间
    /// </summary>
    private DateTimeOffset? _shutdownStartTime;
    
    /// <summary>
    /// 关闭阶段
    /// </summary>
    private ShutdownPhase _currentPhase;
    
    /// <summary>
    /// 取消令牌
    /// </summary>
    private CancellationTokenSource _cts;
    
    // ============================================================
    // 事件
    // ============================================================
    
    /// <summary>
    /// 关闭开始事件
    /// </summary>
    public event Action<ShutdownPhase> OnShutdownPhaseChanged;
    
    /// <summary>
    /// 关闭完成事件
    /// </summary>
    public event Action<int> OnShutdownCompleted;
    
    // ============================================================
    // 公共方法
    // ============================================================
    
    /// <summary>
    /// 注册关闭钩子
    /// </summary>
    public void RegisterShutdownHooks()
    {
        // Windows 关闭事件
        Console.CancelKeyPress += OnCancelKeyPress;
        
        // Linux 信号处理
        AppDomain.CurrentDomain.ProcessExit += OnProcessExit;
        
        // 可选：Windows Service 关闭
        if (OperatingSystem.IsWindows())
        {
            // 使用 SetConsoleCtrlHandler 处理 Windows 控制台事件
        }
        
        Logger.Info("关闭钩子已注册");
    }
    
    /// <summary>
    /// 请求关闭
    /// </summary>
    /// <param name="exitCode">退出码</param>
    public async Task RequestShutdownAsync(int exitCode = 0)
    {
        if (_isShuttingDown)
        {
            Logger.Warning("关闭请求已在处理中，忽略重复请求");
            return;
        }
        
        _isShuttingDown = true;
        _shutdownStartTime = DateTimeOffset.UtcNow;
        _cts = new CancellationTokenSource();
        
        Logger.Info("==========================================");
        Logger.Info("服务器收到关闭请求，开始优雅关闭...");
        Logger.Info($"退出码: {exitCode}");
        Logger.Info("==========================================");
        
        try
        {
            await ExecuteShutdownAsync(exitCode, _cts.Token);
        }
        catch (OperationCanceledException)
        {
            Logger.Warning("关闭操作被取消");
            exitCode = -1;
        }
        catch (Exception ex)
        {
            Logger.Error(ex, "关闭过程中发生错误");
            exitCode = -1;
        }
        finally
        {
            OnShutdownCompleted?.Invoke(exitCode);
            
            Logger.Info("==========================================");
            Logger.Info($"服务器已关闭，退出码: {exitCode}");
            Logger.Info("==========================================");
            
            Environment.Exit(exitCode);
        }
    }
    
    /// <summary>
    /// 执行关闭流程
    /// </summary>
    private async Task ExecuteShutdownAsync(int exitCode, CancellationToken ct)
    {
        // ========================================================
        // 阶段1：准备关闭
        // ========================================================
        
        SetPhase(ShutdownPhase.Preparing, "准备关闭");
        
        // 通知监控系统
        await NotifyMonitoringAsync("SHUTDOWN_START");
        
        // ========================================================
        // 阶段2：停止接受新连接
        // ========================================================
        
        SetPhase(ShutdownPhase.StoppingNewConnections, "停止接受新连接");
        
        // 停止监听
        await Server.StopListeningAsync(ct);
        
        // 向玩家发送维护通知
        await BroadcastMaintenanceNoticeAsync("服务器即将维护，请及时下线保存数据", ct);
        
        Logger.Info("已停止接受新连接");
        
        // 等待一小段时间让通知送达
        await Task.Delay(TimeSpan.FromSeconds(3), ct);
        
        // ========================================================
        // 阶段3：等待消息队列清空
        // ========================================================
        
        SetPhase(ShutdownPhase.DrainingQueue, "等待消息队列清空");
        
        var queueDrained = await DrainMessageQueueAsync(QueueDrainTimeout, ct);
        
        if (queueDrained)
        {
            Logger.Info("消息队列已清空");
        }
        else
        {
            Logger.Warning($"消息队列未在 {QueueDrainTimeout.TotalSeconds} 秒内清空，强制继续");
        }
        
        // ========================================================
        // 阶段4：等待请求处理完成
        // ========================================================
        
        SetPhase(ShutdownPhase.WaitingForRequests, "等待请求处理完成");
        
        var requestsCompleted = await WaitForRequestsCompletionAsync(
            RequestCompletionTimeout, ct);
        
        if (requestsCompleted)
        {
            Logger.Info("所有请求处理完成");
        }
        else
        {
            Logger.Warning($"有 {Server.ActiveRequestCount} 个请求未完成，强制继续");
        }
        
        // ========================================================
        // 阶段5：保存数据
        // ========================================================
        
        SetPhase(ShutdownPhase.SavingData, "保存数据");
        
        await SaveCriticalDataAsync(ct);
        
        Logger.Info("关键数据已保存");
        
        // ========================================================
        // 阶段6：断开所有连接
        // ========================================================
        
        SetPhase(ShutdownPhase.ClosingConnections, "断开所有连接");
        
        await Server.CloseAllConnectionsAsync();
        
        Logger.Info("所有连接已关闭");
        
        // ========================================================
        // 阶段7：清理资源
        // ========================================================
        
        SetPhase(ShutdownPhase.CleaningUp, "清理资源");
        
        await CleanupResourcesAsync();
        
        // ========================================================
        // 阶段8：完成
        // ========================================================
        
        SetPhase(ShutdownPhase.Completed, "关闭完成");
        
        // 刷新日志
        LogManager.Shutdown();
        
        await NotifyMonitoringAsync("SHUTDOWN_COMPLETE");
    }
    
    // ============================================================
    // 辅助方法
    // ============================================================
    
    /// <summary>
    /// 清空消息队列
    /// </summary>
    private async Task<bool> DrainMessageQueueAsync(
        TimeSpan timeout, 
        CancellationToken ct)
    {
        var deadline = DateTimeOffset.UtcNow.Add(timeout);
        
        while (DateTimeOffset.UtcNow < deadline)
        {
            ct.ThrowIfCancellationRequested();
            
            var queueLength = Server.GetMessageQueueLength();
            
            if (queueLength == 0)
            {
                return true;
            }
            
            Logger.Info($"消息队列剩余: {queueLength} 条，等待清空...");
            
            await Task.Delay(TimeSpan.FromSeconds(1), ct);
        }
        
        return false;
    }
    
    /// <summary>
    /// 等待请求处理完成
    /// </summary>
    private async Task<bool> WaitForRequestsCompletionAsync(
        TimeSpan timeout,
        CancellationToken ct)
    {
        var deadline = DateTimeOffset.UtcNow.Add(timeout);
        
        while (DateTimeOffset.UtcNow < deadline)
        {
            ct.ThrowIfCancellationRequested();
            
            var activeCount = Server.ActiveRequestCount;
            
            if (activeCount == 0)
            {
                return true;
            }
            
            Logger.Info($"仍有 {activeCount} 个请求处理中...");
            
            await Task.Delay(TimeSpan.FromSeconds(1), ct);
        }
        
        return false;
    }
    
    /// <summary>
    /// 保存关键数据
    /// </summary>
    private async Task SaveCriticalDataAsync(CancellationToken ct)
    {
        Logger.Info("开始保存玩家数据...");
        
        // 保存所有在线玩家数据
        var savedCount = 0;
        var failedCount = 0;
        
        foreach (var player in Server.GetOnlinePlayers())
        {
            ct.ThrowIfCancellationRequested();
            
            try
            {
                await PlayerDataService.SavePlayerDataAsync(player, ct);
                savedCount++;
            }
            catch (Exception ex)
            {
                Logger.Error(ex, $"保存玩家 {player.Id} 数据失败");
                failedCount++;
            }
        }
        
        Logger.Info($"玩家数据保存完成: 成功={savedCount}, 失败={failedCount}");
        
        // 保存服务器状态快照
        await ServerStateService.SaveSnapshotAsync(ct);
        
        Logger.Info("服务器状态快照已保存");
    }
    
    /// <summary>
    /// 清理资源
    /// </summary>
    private async Task CleanupResourcesAsync()
    {
        // 关闭数据库连接
        await Database.CloseAllConnectionsAsync();
        
        // 关闭 Redis 连接
        await Redis.CloseAllConnectionsAsync();
        
        // 释放内存池
        MemoryPool.Shared.Dispose();
        
        // 清理临时文件
        TempFileManager.Cleanup();
        
        Logger.Info("资源清理完成");
    }
    
    /// <summary>
    /// 设置关闭阶段
    /// </summary>
    private void SetPhase(ShutdownPhase phase, string description)
    {
        _currentPhase = phase;
        
        Logger.Info($"关闭阶段: {phase} - {description}");
        
        OnShutdownPhaseChanged?.Invoke(phase);
    }
    
    /// <summary>
    /// 通知监控系统
    /// </summary>
    private Task NotifyMonitoringAsync(string eventType)
    {
        // 向 Prometheus Pushgateway 或其他监控系统发送事件
        return Task.CompletedTask;
    }
    
    /// <summary>
    /// 广播维护通知
    /// </summary>
    private Task BroadcastMaintenanceNoticeAsync(string message, CancellationToken ct)
    {
        var notice = new S2C_ServerNotice
        {
            Type = 2, // 弹窗
            Title = "服务器维护通知",
            Content = message
        };
        
        return Server.BroadcastAsync(notice, ct);
    }
    
    // ============================================================
    // 信号处理
    // ============================================================
    
    private void OnCancelKeyPress(object sender, ConsoleCancelEventArgs e)
    {
        e.Cancel = true; // 阻止默认行为
        Logger.Info("检测到 Ctrl+C，开始关闭...");
        _ = RequestShutdownAsync(0);
    }
    
    private void OnProcessExit(object sender, EventArgs e)
    {
        Logger.Info("检测到进程退出信号");
        _ = RequestShutdownAsync(0);
    }
}

/// <summary>
/// 关闭阶段
/// </summary>
public enum ShutdownPhase
{
    Preparing,
    StoppingNewConnections,
    DrainingQueue,
    WaitingForRequests,
    SavingData,
    ClosingConnections,
    CleaningUp,
    Completed
}
```

```csharp
/// - 需要重启：端口号、网络协议
/// </summary>
public class HotReloadManager
{
    // ============================================================
    // 字段
    // ============================================================
    
    /// <summary>
    /// 配置文件路径
    /// </summary>
    private readonly string _configPath;
    
    /// <summary>
    /// 文件监控器
    /// </summary>

    /// <summary>
    /// 当前配置
    /// </summary>
    private ServerConfig _currentConfig;
    
    /// <summary>
    /// 配置版本
    /// </summary>
    private int _configVersion;
    
    /// <summary>
    /// 锁
    /// </summary>
    private readonly SemaphoreSlim _lock = new(1, 1);
    
    // ============================================================
    // 事件
    // ============================================================
    
    /// <summary>
    /// 配置变更事件
    /// </summary>
    public event Action<ServerConfig, ServerConfig> OnConfigChanged;
    
    // ============================================================
    // 初始化
    // ============================================================
    
    /// <summary>
    /// 构造函数
    /// </summary>
    public HotReloadManager(string configPath)
    {
        _configPath = configPath;
        
        // 加载初始配置
        _currentConfig = LoadConfig();
        
        // 创建文件监控器
        var directory = Path.GetDirectoryName(_configPath);
        var fileName = Path.GetFileName(_configPath);
        
        _watcher = new FileSystemWatcher(directory, fileName)
        {
            NotifyFilter = NotifyFilters.LastWrite | NotifyFilters.Size,
            EnableRaisingEvents = true
        };
        
        _watcher.Changed += OnConfigFileChanged;
        
        Logger.Info($"配置热更新已启用，监控文件: {_configPath}");
    }
    
    // ============================================================
    // 公共方法
    // ============================================================
    
    /// <summary>
    /// 获取当前配置
    /// </summary>
    public ServerConfig GetConfig()
    {
        return _currentConfig;
    }
    
    /// <summary>
    /// 获取配置版本
    /// </summary>
    public int GetConfigVersion()
    {
        return _configVersion;
    }
    
    /// <summary>
    /// 重新加载配置
    /// </summary>
    public async Task ReloadAsync()
    {
        await _lock.WaitAsync();
        try
        {
            var oldConfig = _currentConfig;
            var newConfig = LoadConfig();
            
            if (oldConfig.Equals(newConfig))
            {
                Logger.Debug("配置未变化，跳过更新");
                return;
            }
            
            Logger.Info("==========================================");
            Logger.Info("检测到配置变化，开始应用新配置...");
            Logger.Info("==========================================");
            
            // 应用配置变更
            await ApplyConfigChangesAsync(oldConfig, newConfig);
            
            _currentConfig = newConfig;
            _configVersion++;
            
            // 触发事件
            OnConfigChanged?.Invoke(oldConfig, newConfig);
            
            Logger.Info("==========================================");
            Logger.Info($"配置更新完成，版本: {_configVersion}");
            Logger.Info("==========================================");
        }
        finally
        {
            _lock.Release();
        }
    }
    
    // ============================================================
    // 配置应用
    // ============================================================
    
    /// <summary>
    /// 应用配置变更
    /// </summary>
    private async Task ApplyConfigChangesAsync(ServerConfig oldConfig, ServerConfig newConfig)
    {
        // ========================================================
        // 日志级别（立即生效）
        // ========================================================
        
        if (oldConfig.LogLevel != newConfig.LogLevel)
        {
            Logger.Info($"日志级别变更: {oldConfig.LogLevel} -> {newConfig.LogLevel}");
            Logger.SetLevel(newConfig.LogLevel);
        }
        
        // ========================================================
        // 帧率配置（下帧生效）
        // ========================================================
        
        if (oldConfig.TargetFps != newConfig.TargetFps)
        {
            Logger.Info($"帧率变更: {oldConfig.TargetFps} -> {newConfig.TargetFps}");
            await GameLoop.SetTargetFpsAsync(newConfig.TargetFps);
        }
        
        // ========================================================
        // 消息队列大小（下帧生效）
        // ========================================================
        
        if (oldConfig.MaxQueueLength != newConfig.MaxQueueLength)
        {
            Logger.Info($"消息队列大小变更: {oldConfig.MaxQueueLength} -> {newConfig.MaxQueueLength}");
            MessageQueue.SetMaxLength(newConfig.MaxQueueLength);
        }
        
        // ========================================================
        // 心跳配置（立即生效）
        // ========================================================
        
        if (oldConfig.HeartbeatInterval != newConfig.HeartbeatInterval)
        {
            Logger.Info($"心跳间隔变更: {oldConfig.HeartbeatInterval}s -> {newConfig.HeartbeatInterval}s");
            HeartbeatManager.SetInterval(newConfig.HeartbeatInterval);
        }
        
        // ========================================================
        // 反外挂阈值（立即生效）
        // ========================================================
        
        if (oldConfig.MaxMoveSpeed != newConfig.MaxMoveSpeed)
        {
            Logger.Info($"最大移动速度变更: {oldConfig.MaxMoveSpeed} -> {newConfig.MaxMoveSpeed}");
            MovementValidator.SetMaxSpeed(newConfig.MaxMoveSpeed);
        }
        
        // ========================================================
        // 告警阈值（立即生效）
        // ========================================================
        
        await UpdateAlertThresholdsAsync(oldConfig, newConfig);
    }
    
    // ============================================================
    // 文件监控
    // ============================================================
    
    /// <summary>
    /// 配置文件变化回调
    /// </summary>
    private void OnConfigFileChanged(object sender, FileSystemEventArgs e)
    {
        // 防抖：延迟一下再处理，等文件写完
        Task.Delay(500).ContinueWith(_ =>
        {
            Logger.Info($"检测到配置文件变化: {e.FullPath}");
            _ = ReloadAsync();
        });
    }
    
    // ============================================================
    // 配置加载
    // ============================================================
    
    /// <summary>
    /// 加载配置
    /// </summary>
    private ServerConfig LoadConfig()
    {
        try
        {
            var json = File.ReadAllText(_configPath);
            return JsonSerializer.Deserialize<ServerConfig>(json);
        }
        catch (Exception ex)
        {
            Logger.Error(ex, $"加载配置文件失败: {_configPath}，使用默认配置");
            return ServerConfig.Default;
        }
    }
    
    /// <summary>
    /// 更新告警阈值
    /// </summary>
    private Task UpdateAlertThresholdsAsync(ServerConfig oldConfig, ServerConfig newConfig)
    {
        if (oldConfig.CpuAlertThreshold != newConfig.CpuAlertThreshold)
        {
            AlertManager.UpdateThreshold("high_cpu", newConfig.CpuAlertThreshold);
        }
        
        if (oldConfig.MemoryAlertThreshold != newConfig.MemoryAlertThreshold)
        {
            AlertManager.UpdateThreshold("high_memory", newConfig.MemoryAlertThreshold);
        }
        
        return Task.CompletedTask;
    }
}

/// <summary>

public class ServerConfig
```

### 7.3 日志文件管理

```csharp
/// <summary>
/// 日志文件管理器
/// 
/// 职责：
/// 1. 按日期/大小切分日志文件
/// 2. 压缩旧日志
/// 3. 自动清理过期日志
/// 4. 多目录输出（本地 + 远程）
/// </summary>
public class LogFileManager
{
    // ============================================================
    // 配置
    // ============================================================
    
    /// <summary>
    /// 日志目录
    /// </summary>
    private readonly string _logDirectory;
    
    /// <summary>
    /// 日志文件大小上限（MB）
    /// </summary>
    private const long MaxFileSizeBytes = 100 * 1024 * 1024; // 100MB
    
    /// <summary>
    /// 日志保留天数
    /// </summary>
    private const int RetentionDays = 30;
    
    /// <summary>
    /// 当前写入文件
    /// </summary>
    private string _currentFile;
    
    /// <summary>
    /// 当前文件大小
    /// </summary>
    private long _currentFileSize;
    
    /// <summary>
    /// 锁
    /// </summary>
    private readonly SemaphoreSlim _writeLock = new(1, 1);
    
    // ============================================================
    // 初始化
    // ============================================================
    
    public LogFileManager(string logDirectory)
    {
        _logDirectory = logDirectory;
        
        // 创建日志目录
        Directory.CreateDirectory(_logDirectory);
        
        // 打开今天的日志文件
        OpenNewFile();
        
        // 启动清理任务
        _ = CleanupOldLogsAsync();
    }
    
    // ============================================================
    // 写入
    // ============================================================
    
    /// <summary>
    /// 写入日志
    /// </summary>
    public async Task WriteAsync(LogEntry entry)
    {
        await _writeLock.WaitAsync();
        try
        {
            // 检查是否需要切换文件
            if (_currentFileSize >= MaxFileSizeBytes)
            {
                RotateFile();
            }
            
            // 格式化
            var formatter = new StructuredLogFormatter();
            var line = formatter.Format(entry) + "\n";
            
            // 写入
            await File.AppendAllTextAsync(_currentFile, line);
            _currentFileSize += Encoding.UTF8.GetByteCount(line);
        }
        finally
        {
            _writeLock.Release();
        }
    }
    
    // ============================================================
    // 文件滚动
    // ============================================================
    
    /// <summary>
    /// 打开新文件
    /// </summary>
    private void OpenNewFile()
    {
        var timestamp = DateTimeOffset.UtcNow.ToString("yyyyMMdd_HHmmss");
        _currentFile = Path.Combine(_logDirectory, $"mmorpg_{timestamp}.json");
        _currentFileSize = 0;
        
        Logger.Info($"打开新的日志文件: {_currentFile}");
    }
    
    /// <summary>
    /// 滚动文件
    /// </summary>
    private void RotateFile()
    {
        // 压缩旧文件
        var oldFile = _currentFile;
        var compressedFile = oldFile + ".gz";
        
        Task.Run(async () =>
        {
            try
            {
                CompressFile(oldFile, compressedFile);
                File.Delete(oldFile);
                Logger.Info($"日志文件已压缩: {oldFile} -> {compressedFile}");
            }
            catch (Exception ex)
            {
                Logger.Error(ex, $"日志文件压缩失败: {oldFile}");
            }
        });
        
        // 打开新文件
        OpenNewFile();
    }
    
    // ============================================================
    // 清理
    // ============================================================
    
    /// <summary>
    /// 清理过期日志
    /// </summary>
    private async Task CleanupOldLogsAsync()
    {
        while (true)
        {
            try
            {
                var cutoffDate = DateTimeOffset.UtcNow.AddDays(-RetentionDays);
                
                var files = Directory.GetFiles(_logDirectory, "mmorpg_*.json*");
                
                foreach (var file in files)
                {
                    var fileInfo = new FileInfo(file);
                    
                    if (fileInfo.LastWriteTimeUtc < cutoffDate)
                    {
                        File.Delete(file);
                        Logger.Info($"删除过期日志文件: {file}");
                    }
                }
            }
            catch (Exception ex)
            {
                Logger.Error(ex, "清理过期日志失败");
            }
            
            // 每天执行一次
            await Task.Delay(TimeSpan.FromDays(1));
        }
    }
    
    /// <summary>
    /// 压缩文件
    /// </summary>
    private void CompressFile(string source, string destination)
    {
        using var sourceStream = File.OpenRead(source);
        using var destStream = File.Create(destination);
        using var compressor = new GZipStream(destStream, CompressionLevel.Optimal);
        sourceStream.CopyTo(compressor);
    }
}
```

## 9. 指标监控系统 (Metrics)

### 9.1 设计目标

- **三类核心指标**：Counter（累计计数）、Gauge（瞬时值）、Histogram（分布统计）
- **高性能**：指标操作使用 Interlocked 原子操作，无锁，零 GC
- **线程安全**：多线程环境下的指标注册、更新、查询均为线程安全
- **可扩展**：通过 IMetric 接口支持自定义指标类型

### 9.2 指标类型

| 类型 | 用途 | 示例 |
|------|------|------|
| Counter | 累计计数 | TPS、消息处理数、错误次数 |
| Gauge | 瞬时值 | 在线会话数、内存使用、CPU 使用率 |
| Histogram | 分布统计 | 消息延迟、序列化时间、GC 暂停时间 |

### 9.3 MetricsCollector 单例

```csharp
// 注册指标
MetricsCollector.Instance.RegisterCounter("server.rps");
MetricsCollector.Instance.RegisterGauge("server.online_sessions");
MetricsCollector.Instance.RegisterHistogram("server.message_latency_ms");

// 更新指标
MetricsCollector.Instance.IncrementCounter("server.rps", 1);
MetricsCollector.Instance.SetGauge("server.online_sessions", 1024);
MetricsCollector.Instance.RecordHistogram("server.message_latency_ms", 23.5);

// 查询指标
var counterValue = MetricsCollector.Instance.GetCounter("server.rps");
var snapshot = MetricsCollector.Instance.GetCounterSnapshot();

// 重置
MetricsCollector.Instance.ResetAll();
```

### 9.4 关键指标建议

```csharp
// 业务指标
MetricsCollector.Instance.RegisterCounter("server.rps");          // 每秒请求数
MetricsCollector.Instance.RegisterCounter("server.message_count");   // 累计消息数

// 性能指标

// 资源指标

MetricsCollector.Instance.RegisterGauge("server.memory_mb");       // 内存使用(MB)
MetricsCollector.Instance.RegisterGauge("server.gc_count");        // GC 次数
```

## 10. 健康检查系统 (Health Check)

### 10.1 设计目标

- **可插拔检查项**：支持注册任意健康检查项
- **聚合报告**：所有检查项结果汇总为一个 JSON/纯文本报告
- **三种状态**：Healthy（健康）、Degraded（降级）、Unhealthy（异常）

### 10.2 HealthCheckService 单例

```csharp
// 注册检查项
HealthCheckService.Instance.RegisterCheck("tcp_listener", () => 
    tcpServer.IsRunning 
        ? HealthCheckResult.Healthy("TCP 监听正常")
        : HealthCheckResult.Unhealthy("TCP 监听异常"));

HealthCheckService.Instance.RegisterCheck("online_sessions", () =>
{
    var count = sessionManager.SessionCount;
    return count > 100000 
        ? HealthCheckResult.Unhealthy($"在线会话数过高: {count}") 
        : HealthCheckResult.Healthy($"在线会话数: {count}");
});

HealthCheckService.Instance.RegisterCheck("memory_usage", () =>
{
    var memoryMB = GC.GetTotalMemory(false) / (1024 * 1024);
    return memoryMB > 8192 
        ? HealthCheckResult.Degraded($"内存使用较高: {memoryMB} MB") 
        : HealthCheckResult.Healthy($"内存使用: {memoryMB} MB");
});

// 执行检查
var report = HealthCheckService.Instance.CheckAll();

// 输出报告
string jsonReport = HealthCheckService.Instance.GetReportAsJson();
string plainReport = HealthCheckService.Instance.GetReportAsPlainText();
```
