### 2.3 网络输出（NetworkSink）

#### 2.3.1 用途说明

NetworkSink 用于将日志发送到远程集中式日志服务器（如 ELK、Loki、自建日志中心），支持生产环境的日志聚合与分析。

#### 2.3.2 启用方式

通过 `LogOptions.EnableNetwork = true` 和 `NetworkEndpoint = "http://logserver:8080/log"` 启用。

#### 2.3.3 工作原理

- 日志先加入内部队列（ConcurrentQueue，容量 10000 条）
- 后台线程按批发送（每批 50 条，或 1 秒超时）
- 使用 HTTP POST 发送 JSON 数组
- 失败时按指数退避重试（100ms → 200ms → 400ms，最多 3 次）
- 超过容量或全部重试失败的日志会被丢弃，避免内存泄漏

#### 2.3.4 注意事项

- 发送失败不会影响业务逻辑（静默处理，每 100 次失败输出一次提示）
- 默认不启用，主要用于生产环境
- 日志条目需可被 System.Text.Json 序列化

#### 2.3.5 代码示例

```csharp
Logger.Initialize(new LogOptions
{
    MinLevel = LogLevel.Info,
    EnableConsole = true,
    EnableFile = true,
    EnableNetwork = true,
    NetworkEndpoint = "http://logserver:8080/log",
    LogDirectory = "logs"
});
```

## 3. 日志格式设计

### 3.1 结构化日志格式

```csharp
/// <summary>
/// 日志条目结构
/// 
/// 所有日志都会转换为这个结构
/// 然后由 Formatter 序列化为字符串
/// </summary>
public class LogEntry
{
    // ---------- 必需字段 ----------
    
    /// <summary>
    /// 时间戳（ISO 8601 格式）
    /// </summary>
    public DateTimeOffset Timestamp { get; init; }
    
    /// <summary>
    /// 日志级别
    /// </summary>
    public LogLevel Level { get; init; }
    
    /// <summary>
    /// 消息模板
    /// </summary>
    public string Message { get; init; }
    
    /// <summary>
    /// 格式化后的消息（实际使用位置参数格式化）
    /// </summary>
    public string FormattedMessage { get; set; }
    
    // ---------- 可选字段 ----------
    
    /// <summary>
    /// 异常信息
    /// </summary>
    public Exception? Exception { get; init; }
    
    /// <summary>
    /// 日志来源
    /// </summary>
    public string Source { get; init; }
    
    /// <summary>
    /// 追踪 ID
    /// </summary>
    public string? TraceId { get; init; }
    
    /// <summary>
    /// 玩家 ID（如果适用）
    /// </summary>
    public long? PlayerId { get; init; }
    
    /// <summary>
    /// 会话 ID
    /// </summary>
    public long? SessionId { get; init; }
    
    /// <summary>
    /// 消息类型（如果适用）
    /// </summary>
    public int? MessageType { get; init; }
    
    /// <summary>
    /// 自定义属性
    /// </summary>
    public Dictionary<string, object?>? Properties { get; init; }
    
    // ---------- 方法 ----------
    
    /// <summary>
    /// 序列化为 JSON 格式（手动构建，避免异常序列化问题）
    /// </summary>
    public string ToJson()
    {
        var parts = new List<string>
        {
            $"\"timestamp\": \"{Timestamp:O}\"",
            $"\"level\": \"{Level}\"",
            $"\"source\": \"{EscapeJson(Source)}\"",
            $"\"message\": \"{EscapeJson(FormattedMessage)}\"",
            $"\"messageTemplate\": \"{EscapeJson(Message)}\""
        };
        if (Exception != null)
            parts.Add($"\"exception\": \"{EscapeJson(Exception.ToString())}\"");
        if (!string.IsNullOrEmpty(TraceId))
            parts.Add($"\"traceId\": \"{EscapeJson(TraceId)}\"");
        if (PlayerId.HasValue)
            parts.Add($"\"playerId\": {PlayerId.Value}");
        if (SessionId.HasValue)
            parts.Add($"\"sessionId\": {SessionId.Value}");

        return "{" + string.Join(", ", parts) + "}";
    }
    
    private static string EscapeJson(string value)
    {
        if (string.IsNullOrEmpty(value)) return "";
        return value.Replace("\\", "\\\\").Replace("\"", "\\\"")
            .Replace("\n", "\\n").Replace("\r", "\\r").Replace("\t", "\\t");
    }
    
    /// <summary>
    /// 转换为控制台友好格式
    /// </summary>
    public string ToConsoleString()
    {
        var parts = new List<string>
        {
            $"[{Timestamp:yyyy-MM-dd HH:mm:ss.fff}]",
            $"[{Level.ToString().ToUpper().PadRight(5)}]",
            $"[{Source.PadRight(15)}]",
            Message
        };
        if (Properties?.Count > 0)
        {
            var props = string.Join(", ", Properties.Select(p => $"{p.Key}={p.Value}"));
            parts.Add($"[{props}]");
        }
        return string.Join(" ", parts);
    }
}

/// <summary>
/// 日志级别枚举
/// </summary>
public enum LogLevel
{
    Trace = 0,
    Debug = 1,
    Info = 2,
    Warning = 3,
    Error = 4,
    Fatal = 5
}
```

## 4. 日志记录器实现

### 4.1 核心Logger类

```csharp
/// <summary>
/// 日志记录器
/// 
/// 核心特性：
/// 1. 静态类，全局可用
/// 2. 支持模板字符串（类似 ILogger）
/// 3. 高性能异步写入
/// 4. 零分配（使用 Span 和 ArrayPool）
/// 
/// 使用示例：
/// ```csharp
/// Logger.Info("Server", "服务器启动完成");
/// Logger.Info("Game", "玩家 {0} 登录成功", playerId);
/// Logger.Warning("Config", "配置项 {0} 缺失，使用默认值 {1}", key, def);
/// Logger.Error("Database", ex, "保存玩家数据失败: PlayerId={0}", playerId);
/// ```
/// </summary>
public static class Logger
{
    // ============================================================
    // 静态字段（全局配置）
    // ============================================================
    
    /// <summary>
    /// 当前日志级别
    /// </summary>
    private static LogLevel _minLevel = LogLevel.Debug;
    
    /// <summary>
    /// 输出目标列表
    /// </summary>
    private static readonly List<ILogSink> _sinks = new();
    
    /// <summary>
    /// 异步写入队列
    /// </summary>
    private static Channel<LogEntry>? _queue;
    
    /// <summary>
    /// 后台写入任务
    /// </summary>
    private static Task? _writeTask;
    
    /// <summary>
    /// 是否已初始化
    /// </summary>
    private static bool _initialized;
    
    /// <summary>
    /// 初始化锁
    /// </summary>
    private static readonly object _initLock = new();
    
    // ============================================================
    // 初始化
    // ============================================================
    
    /// <summary>
    /// 初始化日志系统
    /// </summary>
    /// <param name="options">日志配置</param>
    public static void Initialize(LogOptions options)
    {
        lock (_initLock)
        {
            if (_initialized)
            {
                Warning("Logger", "日志系统已初始化，忽略重复初始化");
                return;
            }
            
            // 设置最小级别
            _minLevel = options.MinLevel;
            
            // 创建异步队列（队列满时等待，而非丢弃）
            _queue = Channel.CreateBounded<LogEntry>(new BoundedChannelOptions(10000)
            {
                SingleReader = true,
                SingleWriter = false,
                FullMode = BoundedChannelFullMode.Wait
            });
            
            // 创建输出目标
            if (options.EnableConsole)
            {
                _sinks.Add(new ConsoleSink());
            }
            if (options.EnableFile)
            {
                _sinks.Add(new FileSink(options.LogDirectory,
                    options.MaxFileSize, options.RetentionDays));
            }
            
            // 启动后台写入任务
            _writeTask = WriteLoopAsync();
            
            // 写入启动日志（使用位置参数 {0}）
            Info("Logger", "日志系统初始化完成，最小级别: {0}", _minLevel);
            
            _initialized = true;
        }
    }
    
    // ============================================================
    // 公共方法
    // ============================================================
    
    /// <summary>
    /// 记录 Trace 日志
    /// </summary>
    [Conditional("TRACE")]
    public static void Trace(string source, string message, params object[] args)
    {
        Log(LogLevel.Trace, source, message, args);
    }
    
    /// <summary>
    /// 记录 Debug 日志
    /// </summary>
    [Conditional("DEBUG")]
    public static void Debug(string source, string message, params object[] args)
    {
        Log(LogLevel.Debug, source, message, args);
    }
    
    /// <summary>
    /// 记录 Info 日志
    /// </summary>
    public static void Info(string source, string message, params object[] args)
    {
        Log(LogLevel.Info, source, message, args);
    }
    
    /// <summary>
    /// 记录 Warning 日志
    /// </summary>
    public static void Warning(string source, string message, params object[] args)
    {
        Log(LogLevel.Warning, source, message, args);
    }
    
    /// <summary>
    /// 记录 Error 日志（带异常）
    /// </summary>
    public static void Error(string source, Exception? exception, string message,
        params object[] args)
    {
        Log(LogLevel.Error, source, message, args, exception);
    }
    
    /// <summary>
    /// 记录 Error 日志
    /// </summary>
    public static void Error(string source, string message, params object[] args)
    {
        Log(LogLevel.Error, source, message, args, null);
    }
    
    /// <summary>
    /// 记录 Fatal 日志（带异常）
    /// </summary>
    public static void Fatal(string source, Exception? exception, string message,
        params object[] args)
    {
        Log(LogLevel.Fatal, source, message, args, exception);
    }
    
    /// <summary>
    /// 记录 Fatal 日志
    /// </summary>
    public static void Fatal(string source, string message, params object[] args)
    {
        Log(LogLevel.Fatal, source, message, args, null);
    }
    
    // ============================================================
    // 内部方法
    // ============================================================
    
    /// <summary>
    /// 核心日志方法（使用位置参数 {0} {1} {2}，不解析命名占位符）
    /// </summary>
    private static void Log(LogLevel level, string source, string message,
        object[] args, Exception? exception = null)
    {
        // 级别检查
        if (level < _minLevel)
            return;
        
        // 确保已初始化
        if (!_initialized || _queue == null)
        {
            Console.Error.WriteLine($"[ERROR] 日志系统未初始化: {source} - {message}");
            return;
        }
        
        // 格式化消息（仅使用位置参数）
        var formattedMessage = args.Length > 0
            ? string.Format(message, args)
            : message;
        
        // 创建日志条目
        var entry = new LogEntry(level, source, message, formattedMessage, exception)
        {
            TraceId = TraceContext.Current?.TraceId,
            PlayerId = PlayerContext.Current?.PlayerId,
            SessionId = PlayerContext.Current?.SessionId
        };
        
        // 尝试写入队列
        if (!_queue.Writer.TryWrite(entry))
        {
            Console.Error.WriteLine($"[WARNING] 日志队列已满，丢弃消息: {formattedMessage}");
        }
    }
    
    /// <summary>
    /// 后台写入循环
    /// </summary>
    private static async Task WriteLoopAsync()
    {
        if (_queue == null)
            return;
        
        await foreach (var entry in _queue.Reader.ReadAllAsync())
        {
            try
            {
                // 并行写入所有输出目标
                var tasks = _sinks.Select(sink => sink.WriteAsync(entry));
                await Task.WhenAll(tasks);
            }
            catch (Exception ex)
            {
                // 防止日志系统异常导致程序崩溃
                Console.Error.WriteLine($"日志写入失败: {ex.Message}");
            }
        }
    }
    
    /// <summary>
    /// 关闭日志系统
    /// </summary>
    public static async Task ShutdownAsync()
    {
        if (!_initialized || _queue == null)
            return;
        
        // 标记队列不再接收新消息
        _queue.Writer.Complete();
        
        // 等待后台任务完成
        if (_writeTask != null)
        {
            try
            {
                await _writeTask.WaitAsync(TimeSpan.FromSeconds(5));
            }
            catch (TimeoutException)
            {
                Error("Logger", "日志系统关闭超时");
            }
        }
        
        // 关闭所有输出
        foreach (var sink in _sinks)
        {
            await sink.DisposeAsync();
        }
        
        _sinks.Clear();
        _initialized = false;
        
        Info("Logger", "日志系统已关闭");
    }
}

/// <summary>
/// 日志输出接口
/// </summary>
public interface ILogSink : IAsyncDisposable
{
    Task WriteAsync(LogEntry entry);
}

/// <summary>
/// 控制台输出（带颜色，使用 lock 保护并发写入）
/// </summary>
public class ConsoleSink : ILogSink
{
    private readonly object _lock = new();

    private static readonly Dictionary<LogLevel, ConsoleColor> Colors = new()
    {
        [LogLevel.Trace] = ConsoleColor.DarkGray,
        [LogLevel.Debug] = ConsoleColor.Gray,
        [LogLevel.Info] = ConsoleColor.White,
        [LogLevel.Warning] = ConsoleColor.Yellow,
        [LogLevel.Error] = ConsoleColor.Red,
        [LogLevel.Fatal] = ConsoleColor.DarkRed
    };

    public Task WriteAsync(LogEntry entry)
    {
        lock (_lock)
        {
            Console.ForegroundColor = Colors.GetValueOrDefault(entry.Level, ConsoleColor.White);
            try
            {
                // 构建日志行（使用 FormattedMessage 而非 Message）
                var parts = new List<string>
                {
                    $"[{entry.Timestamp:yyyy-MM-dd HH:mm:ss.fff}]",
                    $"[{entry.Level.ToString().ToUpper().PadRight(5)}]",
                    $"[{entry.Source.PadRight(15)}]",
                    entry.FormattedMessage
                };
                if (entry.Properties?.Count > 0)
                {
                    var props = string.Join(", ", entry.Properties.Select(p => $"{p.Key}={p.Value}"));
                    parts.Add($"[{props}]");
                }
                if (!string.IsNullOrEmpty(entry.TraceId))
                {
                    parts.Add($"[TraceId={entry.TraceId}]");
                }
                if (entry.PlayerId.HasValue)
                {
                    parts.Add($"[PlayerId={entry.PlayerId}]");
                }

                Console.WriteLine(string.Join(" ", parts));

                if (entry.Exception != null)
                {
                    Console.WriteLine(entry.Exception.ToString());
                }
            }
            finally
            {
                Console.ResetColor();
            }
        }
        return Task.CompletedTask;
    }

    public ValueTask DisposeAsync() => default;
}

/// <summary>
/// 文件输出（使用 entry.ToJson()，自动滚动，gzip 压缩旧日志）
/// </summary>
public class FileSink : ILogSink
{
    private readonly string _directory;
    private readonly long _maxFileSize;
    private readonly int _retentionDays;

    private string _currentFile = string.Empty;
    private long _currentFileSize;
    private StreamWriter? _writer;
    private readonly SemaphoreSlim _writeLock = new(1, 1);

    public FileSink(string directory, long maxFileSize, int retentionDays)
    {
        _directory = directory;
        _maxFileSize = maxFileSize;
        _retentionDays = retentionDays;

        Directory.CreateDirectory(_directory);
        OpenNewFile();
        CleanupOldFiles();
    }

    public async Task WriteAsync(LogEntry entry)
    {
        await _writeLock.WaitAsync();
        try
        {
            // 使用 LogEntry.ToJson() 方法
            var line = entry.ToJson();

            // 检查是否需要滚动
            if (_currentFileSize + line.Length > _maxFileSize)
            {
                await RotateFileAsync();
            }

            // 写入
            await _writer!.WriteLineAsync(line);
            await _writer.FlushAsync();
            _currentFileSize += line.Length + Environment.NewLine.Length;
        }
        finally
        {
            _writeLock.Release();
        }
    }

    private void OpenNewFile()
    {
        var timestamp = DateTimeOffset.UtcNow.ToString("yyyyMMdd_HHmmss");
        _currentFile = Path.Combine(_directory, $"mmorpg_{timestamp}.json");
        _writer = new StreamWriter(_currentFile, append: true)
        {
            AutoFlush = false
        };
        _currentFileSize = 0;
    }

    private async Task RotateFileAsync()
    {
        await _writer!.DisposeAsync();
        _ = Task.Run(() => CompressFile(_currentFile));
        OpenNewFile();
    }

    private void CompressFile(string file)
    {
        try
        {
            if (!File.Exists(file))
                return;

            var compressed = file + ".gz";
            using var inputStream = File.OpenRead(file);
            using var outputStream = File.Create(compressed);
            using var gzipStream = new GZipStream(outputStream, CompressionLevel.Optimal);

            inputStream.CopyTo(gzipStream);
            File.Delete(file);

            Logger.Debug("FileSink", "日志文件已压缩: {0} -> {1}", file, compressed);
        }
        catch (Exception ex)
        {
            Logger.Error("FileSink", ex, "日志文件压缩失败: {0}", file);
        }
    }

    private void CleanupOldFiles()
    {
        try
        {
            var cutoff = DateTimeOffset.UtcNow.AddDays(-_retentionDays);
            var files = Directory.GetFiles(_directory, "mmorpg_*.json*");

            foreach (var file in files)
            {
                try
                {
                    var fileInfo = new FileInfo(file);
                    if (fileInfo.LastWriteTimeUtc < cutoff)
                    {
                        File.Delete(file);
                    }
                }
                catch { }
            }
        }
        catch { }
    }

    public async ValueTask DisposeAsync()
    {
        await _writeLock.WaitAsync();
        try
        {
            if (_writer != null)
            {
                await _writer.FlushAsync();
                await _writer.DisposeAsync();
                _writer = null;
            }
        }
        finally
        {
            _writeLock.Release();
        }
    }
}

        return value[..visibleChars] + new string('*', value.Length - visibleChars);

## 5. 网络日志输出 (NetworkSink)

### 5.1 设计目标

- **集中化日志**：将多台游戏服务器的日志发送到中央日志收集服务（如 ELK、Grafana Loki）
- **结构化传输**：以 JSON 格式通过 HTTP POST 发送，便于下游系统解析
- **批量发送**：累积到一定数量或时间后批量发送，降低网络开销
- **指数退避重试**：发送失败后自动重试，采用指数退避策略
- **队列保护**：日志发送队列有容量上限，超过容量丢弃旧日志，防止内存泄漏

### 5.2 NetworkSink 配置

```csharp
var networkSink = new NetworkSink(
    endpoint: "http://log-collector.example.com:8080/api/logs",
    batchSize: 100,          // 每批最多 100 条日志
    flushIntervalMs: 2000,   // 每 2 秒强制刷新一次
    maxQueueSize: 10000);    // 队列最大 10000 条

// 添加到 Logger
logger.AddSink(networkSink, LogLevel.Info);
```

### 5.3 传输格式

```json
[
    {
        "Timestamp": "2026-06-22T12:34:56.789Z",
        "Level": "Info",
        "Category": "Server",
        "Message": "Server started successfully",
        "Properties": {
            "ServerId": "server-01",
            "Version": "1.0.0"
        }
    }
]
```

### 5.4 失败处理策略

```csharp
// 内部实现的失败处理逻辑：
// 1. 立即记录错误到文件 Sink（不丢失关键信息）
// 2. 将失败的日志放回队列，下次重试
// 3. 下一次发送前等待更长时间（指数退避）
// 4. 队列满时丢弃最旧的日志，防止内存无限增长
```

## 6. 日志输出矩阵

| Sink | 适用场景 | 优点 | 缺点 |
|------|---------|------|------|
| ConsoleSink | 开发调试 | 实时查看，颜色区分 | 无持久化 |
| FileSink | 生产环境 | 持久化，便于排查 | 需要磁盘空间 |
| NetworkSink | 多服集群 | 集中化，便于分析 | 需要网络连通 |

---

> **提示**：日志是排查问题的第一手资料。请确保关键操作都有适当的日志记录，同时避免过度日志化影响性能。
