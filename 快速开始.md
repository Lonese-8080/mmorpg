# MMORPG 服务端框架

## 项目状态

🟢 **框架核心已完成** — Network/Threading/Logging/Configuration/Observability/Security/Resilience 模块已实现并通过测试

## 快速开始

### 环境要求

- .NET 10 SDK
- C# 13+
- Protobuf 编译器（protoc）— 用于生成消息代码

### 构建项目

```bash
# 还原依赖
dotnet restore

# 构建
dotnet build

# 运行服务器
dotnet run --project src/MMORPG.Server/MMORPG.Server.csproj
```

### 生成 Protobuf 代码

```bash
# 安装 protoc（Windows）
# 下载 https://github.com/protocolbuffers/protobuf/releases

# 生成 C# 代码
protoc --csharp_out=protos/generated protos/common.proto
protoc --csharp_out=protos/generated protos/game.proto
```

### 运行测试

```bash
dotnet test
```
